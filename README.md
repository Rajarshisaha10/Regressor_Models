📈 Regression Models From Scratch

A complete implementation of core regression algorithms built entirely from scratch using NumPy, without relying on machine learning libraries like Scikit-learn for model training.

This project demonstrates a deep understanding of the mathematics behind regression models, optimization techniques, and loss minimization.

🚀 Project Overview

This repository contains custom implementations of:

Linear Regression (Simple & Multiple)

Gradient Descent Optimization

Mean Squared Error (MSE) Loss

R² Score Evaluation

Feature Scaling (Standardization)

Train-Test Split Logic

All models are implemented using only:

Python
NumPy
Matplotlib (for visualization)

No ML frameworks were used for model building.

🧠 What This Project Demonstrates

Understanding of cost functions

Manual gradient computation

Vectorized implementations

Optimization using gradient descent

Convergence analysis

Model evaluation metrics

This is not just model usage — this is model construction.
